# 🔍 truecrime.wtf

Your one-stop rabbit hole into true crime.

A curated collection of the best true crime podcasts, YouTube channels, Reddit threads, docuseries, unsolved cases, and crime maps — all in one place.

## What's inside

- 🎙️ Podcasts — Serial, Crime Junkie, Desi Crime Podcast & more
- 📺 YouTube — Buzzfeed Unsolved, Truly Criminal, Coffeehouse Crime & more
- 🧵 Reddit — r/UnresolvedMysteries, r/TrueCrime, r/IndianCrimeTales & more
- 📚 Docuseries — Making a Murderer, House of Secrets, Curry & Cyanide & more
- 🔓 Unsolved Cases — NamUs, Doe Network, FBI Most Wanted & more
- 🗺️ Crime Maps & Data — SpotCrime, CrimeWatch, NCRB India & more

## Tech

Plain HTML + CSS. No frameworks, no build step.

## Run locally

Just open `index.html` in a browser.
